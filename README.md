# Task-04: Sudoku Solver
## Overview
This project involves creating a Sudoku Solver that automatically solves Sudoku puzzles. The program accepts an input grid representing an unsolved Sudoku puzzle and uses an algorithm to fill in the missing numbers. It explores possible solutions using backtracking or other suitable techniques to find the correct arrangement of numbers for the puzzle. Once solved, the program displays the completed Sudoku grid.

## Features
Automated Sudoku Solving: The program efficiently solves Sudoku puzzles using backtracking or similar algorithms.
Error Highlighting: Identifies incorrect rows, columns, and subgrids in the puzzle and highlights them with a red border.
User-Friendly Interface: Users can input their Sudoku puzzles and see the solved version with a clear and clean UI.
